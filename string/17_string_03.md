- push_back尾插字符,append尾插字符串
- 尾插一般直接用+=
- insert谨慎使用
- erase可以完成头删尾删,也可以删除指定位置之后的
- replace可以把从pos位置开始的字符替换成指定字符.少替换多向后移，多替换少也需要移动数据,效率不高,需要一次一次替换,可以用空间换时间来提高效率
- find系列可以做到寻找,find正着找，refind倒着找
- find_first_of找匹配到的任意一个字符相等都要返回
- finf_last_of就是find_first_of倒着找
- IO流默认空格和换行进行分割，get_line默认遇到换行才终止，遇到空格会继续存储,还可以指定遇到一个字符再终止，所以这个接口很自由
- 左闭右开才能求区间内的个数
